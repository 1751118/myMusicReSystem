This folder contains the Train/Test splits for the Music Recommendation experiments conducted using Factorization Machines.

The following are the sub-folders contained:

-Context
Contains the training set (train.csv) and test set (test.csv) used for Context-Aware Music Recommendation experiments.
Please note that only the positive samples are included in these files.
These train-test splits have been used to create the datasets that have been used in the random sampling (POP_RND) and user-based sampling (POP_USER) scenarios.

-Sequence
Contains the training set (train_SEQ.csv) and test set (test_SEQ.csv) used for Context-Aware Next-Song Recommendation experiments.
Please note that only the positive samples are included in these files.
